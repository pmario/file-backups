# File Backups was uninstalled

Thanks for trying the extension!

## One small clean-up step

If you have any **TiddlyWiki tabs open right now**, please reload them
(`Ctrl+R` / `F5` / `Cmd+R`).

While the extension was installed it added a few helper tiddlers under
`$:/temp/plugins/file-backups/*` (the "save as" toolbar button, a readme
and a license tiddler). They live in memory only and are **never saved
into your wiki file**, but they stay visible in the open tab until the
page is reloaded.

After reload, no trace of the extension remains in your wiki.

## If you reinstall later

Just reload your TiddlyWiki tab once after re-enabling the extension —
the Save As button reappears and saving works again.
